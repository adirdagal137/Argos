# SNAPSHOT V1_0_X GOOGLE-NATIVE

Fecha de congelacion: `2026-04-09`

Este folder conserva la memoria humana pre-`1.1` de Argos que vivia en Google-native.

Principio:
- se preserva todo para lectura y meta-analisis;
- se detiene su papel como backend canonico;
- la escritura viva sigue solo en `ARGOS_RUNTIME`.

Contiene:
- snapshot estructurado de `ARGOS__root_logs`;
- snapshot estructurado de `ARGOS__shadow_log`;
- mapa de pendientes y criterios que antes residian en pestañas humanas.

Estos archivos son suenos de otras cubiertas: utiles para leer el pasado, inutiles como fuente de verdad presente.
